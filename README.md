# SD-80-keymap

Rainbow Keyboard（あっきー氏制作）で Roland SD-80 / SD-90 を使用するための音色マップファイルです。

## 概要
- Roland SD-80 用に作成していますが、SD-90 でも動作するはずです。
- 音色選択をスムーズに行うための定義ファイルです。

## 動作環境
- **Rainbow Keyboard ver 1.36** 以降

## インストール方法
本リポジトリ内の **SD-80.txt** ファイルを、お使いの Rainbow Keyboard インストールフォルダ内の `rkdata` フォルダへコピーしてください。
（例：`rbkey136/rkdata/` 内に配置）

## 注意事項（重要）
音色を選択する前に、必ず以下のいずれかの方法で **Native ON** を送信してください。
- SD-80 本体のパネル操作で設定
- 他の MIDI ユーティリティソフト等から送信
※ Native ON 状態でなければ、正しく音色が切り替わらない場合があります。
